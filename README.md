
# To-Do List Application

This is a simple Python application done as part of Indian Data Club 30 days Python Challenge.



## Tech Stack

**Programming Language:** Python 3.11.4

**Python Packages:** psycopg2,dotenv,streamlit,pandas

**Web App:** StreamLit

**Database:** Postgresql

**IDE:** VSCODE
